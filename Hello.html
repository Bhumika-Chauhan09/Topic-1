Activity 1

  • What is an IoT Application?

    IoT applications use machine learning techniques to analyse massive amounts of linked sensor data in the cloud. Using real-time IoT dashboards and alerts, you may view significant performance indicators, statistics for the mean time between failures, and other data.


  • What are the main features needed?

    Features are required in an IoT Application are as follows: -

    • Smart parking
    • Driverless cars
    • Smart grids
    • Smart homes
    • Smart agriculture systems
    • Smart restaurant systems

    Some exaples of an IoT Application are as folliows: -
